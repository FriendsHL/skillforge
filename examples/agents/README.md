# Example Agent Definitions

These YAML files are importable via the `skillforge` CLI:

```bash
skillforge agents create -f examples/agents/general-assistant.yaml
skillforge agents create -f examples/agents/calculator-helper.yaml
skillforge agents create -f examples/agents/research-agent.yaml
```

Each file uses the schema documented in the root `README.md` under the
`## CLI` section. Edit freely, version them in git, and share them.
