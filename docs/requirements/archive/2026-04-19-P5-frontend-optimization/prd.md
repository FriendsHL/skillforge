## 摘要

重做 Dashboard 设计系统、布局、导航、关键页面和交互状态。

## 目标

- 建立 CSS custom properties token 字典。
- 重构主要页面体验。
- 提升响应式、a11y、loading/error/empty states。
- 改善 Chat、Traces、Sessions、Skills、Agents 等核心页面。

## 非目标

- 不做全新产品营销页。
- 不引入与项目风格冲突的重型 UI 框架。

## 功能需求

- 主题和设计 token。
- Layout / ActivityRail / CmdK。
- 主要页面视觉和交互重构。
- Traces / Session drawer / Chat right rail / Skills 分类等关键增强。

## 验收标准

- [x] Dashboard 主要页面完成视觉升级。
- [x] 响应式和可访问性得到改善。
- [x] 关键页面可用性提升。

## 验证预期

- Frontend build。
- 关键页面浏览器检查。
