Prototype Coverage

对应 IOS-ASSISTANT-COMPANION V1：先做用户能用的手机助理入口，再把更重的 Share Extension、语音和手机 tool node 放到后续版本。

  1. 1

    Pair

    Dashboard 生成短期 QR，iOS probe endpoints[] 并保存首个可达 endpoint。

  2. 2

    Chat

    配对后直接进入默认助理会话，保留附件和系统键盘听写入口。

  3. 3

    Decide

    ask_user 与 confirmation 做成内联卡片，适合移动端快速处理。

  4. 4

    Notify

    push 只负责唤醒和跳转，真实状态仍由前台 REST catch-up 获取。

  5. 5

    Revoke

    device token 从服务端派生身份，撤销后不能继续访问 session。

Static artifact. 这个页面不连接后端，不保存任何 token；它用于对齐移动端信息架构和首版交互范围。
